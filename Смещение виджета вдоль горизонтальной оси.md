Атрибут `app:layout_constraintHorizontal_bias` в `ConstraintLayout` используется для управления смещением виджета вдоль горизонтальной оси, когда на виджет действуют одновременно две горизонтальные constraints (ограничения).

### Объяснение

`layout_constraintHorizontal_bias` принимает значения от 0 до 1:

- **0**: виджет будет выровнен по левому ограничению (start constraint).
- **1**: виджет будет выровнен по правому ограничению (end constraint).
- **0.5**: виджет будет выровнен по центру между двумя ограничениями.
